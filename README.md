# Tournament Manager 🐍

A professional desktop application designed to manage sports tournaments, built entirely in Python using the **Model-View-Controller (MVC)** architectural pattern. 

## 🛠️ Key Features
- **MVC Architecture:** Structured code separation dividing core logic, UI display, and data controllers for clean maintainability.
- **Data & Statistics Modules:** Built-in managers to handle sports statistics, player tracking, and match coordination.
- **Graphical User Interface (GUI):** A clean, desktop window interface designed using Python's `Tkinter` library.

## 🚀 Technologies Used
- **Language:** Python 3.x
- **UI Framework:** Tkinter
- **Software Patterns:** Object-Oriented Programming (OOP) & MVC

## 💻 Project Structure Overview
- `models/`: Contains core data structures (`joueur.py`, `equipe.py`, `match.py`).
- `views/`: Manages user interface renders (`app.py`).
- `managers/`: Orchestrates the logic between data models and UI views (`dataManager.py`).
